## Global Shortcuts

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Command palette | `Cmd + K` | `Ctrl + K` |
| Search | `Cmd + K` then type | `Ctrl + K` then type |
| Go to Library | `Cmd + 1` | `Ctrl + 1` |
| Go to Notes | `Cmd + 2` | `Ctrl + 2` |
| Go to Drafts | `Cmd + 3` | `Ctrl + 3` |
| Go to Code | `Cmd + 4` | `Ctrl + 4` |

## Editor Shortcuts

### Text Formatting

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Bold | `Cmd + B` | `Ctrl + B` |
| Italic | `Cmd + I` | `Ctrl + I` |
| Strikethrough | `Cmd + Shift + X` | `Ctrl + Shift + X` |
| Inline code | `Cmd + E` | `Ctrl + E` |
| Link | `Cmd + K` | `Ctrl + K` |

### Document Actions

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Save | `Cmd + S` | `Ctrl + S` |
| Undo | `Cmd + Z` | `Ctrl + Z` |
| Redo | `Cmd + Shift + Z` | `Ctrl + Shift + Z` |
| Find | `Cmd + F` | `Ctrl + F` |
| Find & Replace | `Cmd + H` | `Ctrl + H` |

### Headings

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Heading 1 | `Cmd + 1` | `Ctrl + 1` |
| Heading 2 | `Cmd + 2` | `Ctrl + 2` |
| Heading 3 | `Cmd + 3` | `Ctrl + 3` |

## PDF Viewer Shortcuts

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Zoom in | `Cmd + =` | `Ctrl + =` |
| Zoom out | `Cmd + -` | `Ctrl + -` |
| Fit to width | `Cmd + 0` | `Ctrl + 0` |
| Next page | `→` or `Space` | `→` or `Space` |
| Previous page | `←` or `Shift + Space` | `←` or `Shift + Space` |
| Go to page | `Cmd + G` | `Ctrl + G` |

## Navigation

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Back | `Cmd + [` | `Alt + ←` |
| Forward | `Cmd + ]` | `Alt + →` |
| Toggle sidebar | `Cmd + \` | `Ctrl + \` |

## Tips

<Tip>
**Start with the Command Palette**

`Cmd/Ctrl + K` is your universal shortcut. Search for any action, find files, and execute commands without memorizing everything.
</Tip>

### Learning Path

<Steps>
  <Step title="Command Palette">
    Master `Cmd/Ctrl + K` first
  </Step>
  <Step title="Formatting">
    Add `B` for bold, `I` for italic
  </Step>
  <Step title="Navigation">
    Learn `1`, `2`, `3`, `4` for quick section jumps
  </Step>
  <Step title="PDF Viewer">
    Master zoom and page navigation
  </Step>
</Steps>

<Note>
Some shortcuts may vary based on browser and operating system.
</Note>
