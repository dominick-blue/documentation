Export your manuscripts to various formats for sharing, submission, or further editing.

## Export Formats

| Format | Extension | Best For |
|--------|-----------|----------|
| **PDF** | `.pdf` | Sharing, printing, archiving |
| **Word** | `.docx` | Collaboration, track changes |
| **LaTeX** | `.tex` | Journal submission, Overleaf |
| **Markdown** | `.md` | Backup, version control |

## How to Export

<Steps>
  <Step title="Open a draft">
    Select the manuscript you want to export
  </Step>
  <Step title="Click Export">
    Opens the export dialog
  </Step>
  <Step title="Select format">
    Choose PDF, DOCX, LaTeX, or Markdown
  </Step>
  <Step title="Choose citation style">
    Select how your bibliography should be formatted
  </Step>
  <Step title="Export">
    File downloads to your computer
  </Step>
</Steps>

## Format Details

<Tabs>
  <Tab title="PDF">
    Creates a print-ready document:
    - Formatted text and headings
    - Rendered math equations
    - Code blocks with styling
    - Bibliography at the end

    **Use for:** Final submissions, sharing with non-technical readers
  </Tab>
  <Tab title="Word (DOCX)">
    Creates an editable Word document:
    - Full formatting preserved
    - Compatible with Google Docs
    - Track changes ready

    **Use for:** Collaboration with co-authors, institutional requirements
  </Tab>
  <Tab title="LaTeX">
    Creates a `.tex` file:
    - Clean LaTeX source
    - Bibliography in BibTeX format
    - Ready for Overleaf import

    **Use for:** Journal submissions, conference papers, thesis
  </Tab>
  <Tab title="Markdown">
    Creates a plain `.md` file:
    - Portable format
    - Version control friendly
    - Can be re-imported

    **Use for:** Backup, sharing source, archiving
  </Tab>
</Tabs>

## Citation Styles

Choose how citations and bibliography are formatted:

| Style | In-Text | Bibliography |
|-------|---------|--------------|
| **APA 7th** | (Author, 2024) | Full APA format |
| **MLA 9th** | (Author 2024) | MLA format |
| **Chicago 17th** | (Author 2024, 1) | Chicago notes |
| **IEEE** | [1] | Numbered references |
| **Harvard** | (Author 2024) | Harvard style |

## Export Workflows

<AccordionGroup>
  <Accordion title="For Journal Submission">
    1. Write and revise in Dune (Drafts)
    2. Export to **LaTeX**
    3. Import into **Overleaf**
    4. Apply journal template
    5. Final formatting and submission
  </Accordion>
  <Accordion title="For Collaboration">
    1. Write draft in Dune
    2. Export to **DOCX**
    3. Share with co-authors
    4. Use track changes for revisions
    5. Re-export when updates incorporated
  </Accordion>
  <Accordion title="For Archiving">
    1. Complete your manuscript
    2. Export to **PDF** (final version)
    3. Export to **Markdown** (editable backup)
    4. Store both in your archive
  </Accordion>
</AccordionGroup>

## Troubleshooting

<Warning>
**Citations Not Appearing?**
- Ensure papers are imported in Library
- Check citation syntax in document
- Verify citation style is selected
</Warning>

<Note>
**Large Documents:** Export may take a moment for long manuscripts. Check that your browser didn't block the download.
</Note>
