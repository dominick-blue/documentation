Drafts is where you write manuscripts, papers, and documents with integrated citation support.

## Creating a Draft

1. Navigate to **Drafts** in the sidebar
2. Click **New**
3. Enter a title for your manuscript
4. Start writing

New drafts come with a basic structure:

```markdown
# Your Title

## Abstract
Write your abstract here...

## 1. Introduction
Start writing...
```

```
┌──────────────────┬─────────────────────────────────────┐
```

### Header Bar

- **Title** — Current manuscript name
- **Word count** — Live word/character count
- **Version** — Current version number
- **Citation style** — Active style (APA, MLA, etc.)
- **Cite button** — Open citation picker
- **Export button** — Export to various formats

## Focus Mode

For distraction-free writing:

1. Click the **expand icon** in the toolbar
2. Editor goes full-screen
3. Minimal UI — just you and the words
4. Click **Exit Focus** to return

<Tip>
Focus mode is great for first drafts. Exit to check formatting and preview.
</Tip>

## Citations

### Opening the Citation Picker

Three ways to insert a citation:

1. Click the **Cite** button in the toolbar
2. Type `@` and start typing a paper title
3. Use keyboard shortcut (coming soon)

### Citation Picker Dialog

```
┌─────────────────────────────────────────┐
│  Insert Citation                        │
├─────────────────────────────────────────┤
│  Style: [APA] [MLA] [Chicago] [IEEE]    │
│                                         │
│  [Search papers...]                     │
│                                         │
│  Recent:                                │
│  > Smith et al. 2024 - Deep Learning    │
│  > Chen et al. 2023 - Attention         │
└─────────────────────────────────────────┘
```

### Citation Styles

| Style | Example |
|-------|---------|
| **APA** | (Smith, 2024) |
| **MLA** | (Smith 2024) |
| **Chicago** | (Smith 2024, 1) |
| **IEEE** | [1] |

The style applies to all citations in the document.

The editor supports full Markdown plus extensions:

### Headings

```markdown

```

```markdown

```

## Exporting

### Export Formats

| Format | Best For |
|--------|----------|
| **PDF** | Sharing, printing, submission |
| **DOCX** | Editing in Word, collaboration |
| **LaTeX** | Overleaf, journal submission |
| **Markdown** | Backup, version control |

### Export Workflow

<Steps>
  <Step title="Click Export">
    Open the export dialog
  </Step>
  <Step title="Choose format">
    Select PDF, DOCX, LaTeX, or Markdown
  </Step>
  <Step title="Select citation style">
    Choose bibliography format
  </Step>
  <Step title="Download">
    File saves to your computer
  </Step>
</Steps>
